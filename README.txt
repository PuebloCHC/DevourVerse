Devourverse 3D prototype

Files:
- index.html
- style.css
- script.js

How to run:
1. Put all files in the same folder.
2. Open index.html in a modern browser.
3. On phone, tap Start and drag anywhere to summon the dynamic bubble joystick.
4. On desktop, use WASD or arrow keys.

What is included:
- 3D endless eat-to-grow gameplay
- dynamic bubble joystick that expands while moving
- suction-style swallowing for more satisfying eating
- progression into cosmic stages
- black hole collapse and alternate-dimension restart
- total score carries on forever

Notes:
- This is a lightweight prototype designed to be easy to edit.
- You can tune object count, growth speed, suction, and stage thresholds inside script.js.
